🔧 Project Title:
Smart Notes & Todo Management System

🧩 Project Overview:
This is a full-stack web application that allows users to:

Register and log in securely using email verification and JWT-based token authentication.

Manage personal notes with advanced features.

Manage tasks with priorities, statuses, and reminders.

💻 Technology Stack:
Backend:
Spring Boot (REST, JPA, AOP, Security, Actuator, OAuth Login, Logging, Caching, etc.)
Testing Tools: JUnit, Mockito
DevOps & Monitoring: Docker, JMeter, SonarQube, Swagger, Scheduler

Frontend:
Angular or React

Database:
MySQL

Tools:
GitHub, STS (Spring Tool Suite), Figma

Deployment:
AWS

🔐 Key Features:
1. User Authentication and Authorization
User Registration with email verification

JWT-based Login

Role-based Access Control (Admin, User)

2. Notes Management
Create, update, delete categories & notes

Filter notes by category

Attach files (PDF, images) to notes

Download/display attachments

Paginated note view

Search by title, content, etc.

Mark notes as favorite

Export notes to Excel/PDF

Copy notes

3. Todo Management
Create, edit, delete tasks

Set task priority (Low, Medium, High)

Set task status: “To-Do”, “In Progress”, “Completed”

Set reminders (push notifications, emails)
